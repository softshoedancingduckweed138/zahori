# 📺 zahori - Find video streams on any website

[![](https://img.shields.io/badge/Download-Zahori-blue.svg)](https://github.com/softshoedancingduckweed138/zahori)

Zahori helps you find and save media streams from websites. Many sites hide the direct link to a video file to prevent you from saving it. Zahori acts as a browser to look behind those web pages. It identifies the true source of HLS, DASH, or MP4 streams. You can use it to capture videos for offline viewing. It learns how different websites work so it improves as you use it.

3. Follow the prompts on the screen to finish the installation.

## 🛠️ Handling media streams

Once Zahori finds the stream, it displays the available quality options. Select the resolution you prefer. Higher numbers mean better picture quality but larger file sizes. Click the download button next to your chosen option. The software saves the file into your default downloads folder.

## 💡 Troubleshooting common issues

If the application fails to find a video, check your internet connection first. Some websites require you to log in to see content. If these sites block automated tools, Zahori might need to learn the site profile. Use the refresh button to attempt the connection again. If the video does not appear, ensure you copied the full website address from your browser. 

If you encounter an error during the capture phase, check your hard drive space. Sometimes large video files require more room than the application expects. Close other programs if your computer runs slowly during the streaming process. 

## 🛡️ Usage guidelines

Use this software to save content you own or content that allows downloads. Obey the terms of service of the websites you visit. Do not distribute copyrighted material you obtain through this tool. Zahori exists to help with archiving and personal viewing needs. 

## 📝 Configuration notes

The application saves settings automatically. You can change the default save location by going to the settings menu. You can also adjust how many concurrent connections the app uses for downloads. Higher numbers speed up your download but require more bandwidth. Stick to the default settings if you experience connection drops.
